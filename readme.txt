Project Title : Automated Student Performance Analysis System

Developed By  : Shyam
Department    : CSE

Description:
This project automatically reads student data from a CSV file,
calculates performance, predicts at-risk students, generates
recommendations, and saves the final analysis report.

How to Run:
1. Open terminal in the project folder
2. Run: python "analyze student.py"
3. Output will be displayed and saved as final_student_analysis.csv
